# GlowEdge
Glowing Edge CardView Library
Get Started With
Step 1. Add the JitPack repository to your build file
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
Step 2. Add the dependency
dependencies {
	        implementation 'com.github.gaurav-bamugade:GlowEdge:1.0.0'
	}
